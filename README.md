# -Conduct-Risk-Analysis-Tool-Dassault-Aviation
Built a data-driven conduct risk model combining financial, regulatory, and sentiment data to assess corporate risk exposure.

Designed a 7-pillar risk framework (Market Conduct, Compliance, Governance, etc.)
Converted qualitative data into quantitative risk scores using LLM-based sentiment analysis
Developed a weighted scoring model in Excel
Implemented Monte Carlo simulation (2000 scenarios) for stress testing
Built a React.js dashboard for dynamic risk visualization
Created a scalable framework for quantitative risk assessment and scenario analysis
